# 🎮 Online Games – Free Browser-Based HTML, CSS & JavaScript Games

Play a collection of **free online games directly in your browser** — no download required! 🚀
This project is built using **HTML, CSS, and JavaScript**, making it perfect for beginners and web developers.

👉 **Play Now:** https://github.com/Rkgaming606/Onlinegames.git



## 🕹️ About This Project

**Online Games** is a lightweight collection of fun and simple browser games.
It is designed for:

* Beginners learning web development
* Students looking for JavaScript game projects
* Anyone who wants quick, fun games without installation



## ✨ Features

* 🎯 Easy to play games
* ⚡ Fast and lightweight performance
* 🌐 Runs directly in your web browser
* 📱 Beginner-friendly code structure
* 🧠 Great for learning JavaScript game development



## 🎮 How to Play

1. Open the project in your browser
2. Choose a game
3. Follow the on-screen instructions
4. Enjoy playing 😄
